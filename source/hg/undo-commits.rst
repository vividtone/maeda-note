Mercurialでコミットを取り消す
============================================================

直前の1コミットだけ取り消す
------------------------------------------------------------

直前の1コミットだけを取り消すのであれば :command:`hg rollback` を実行すれば良い。

.. code-block:: shell

  hg rollback


複数のコミットをまとめて取り消す
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

:command:`hg strip --keep` を実行すると、指定したリビジョン以降の履歴が破棄される。オプション :command:`--keep` を付けると作業領域は維持される。 :command:`--keep` を付け忘れると履歴の破棄と同時に作業領域も過去の状態に戻るので要注意。

.. code-block:: shell

  hg strip --keep リビジョン
